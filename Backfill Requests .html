About Backfill

Backfill allows you to create leave requests for the teams you manage without the requests going through the approvals process. These requests are created in an approved state - that means there will be no notifications emails to anyone. It also allows you to override whether the request does or does not deduct the time from their absence allowance.

Your name will appear on the audit log as the request creator via backfill. The term "backfill" comes from the expectation that this will allow for the easy creation of previously taken absences, but it can equally be used for future requests.

To get started enter the email address of the user you wish to create backfill requests for. Further information can be found here.
